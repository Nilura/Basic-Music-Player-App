# Basic Music Player App 


A basic music player app, where you can upload your favourite songs and enjoy songs peacefully.




## Dependencies
- file_picker: ^1.9.0+1
- cloud_firestore: ^0.13.5
- firebase_storage: ^3.1.5
- music_player: ^0.1.4
